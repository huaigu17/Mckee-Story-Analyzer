# 格式自适应标准 (Format Adaptation) - 麦基理论的跨格式应用

## 核心认识

麦基《故事》的理论体系建立在对**主流长片剧本**的分析之上（90-120 分钟，对应长篇小说篇幅）。这意味着许多核心标准（五部分结构、人物弧光、多维度对抗力等）预设了足够的叙事空间来展开。

当分析不同篇幅的创作时，**必须调整标准尺度**。用长片标准衡量极短篇，如同用马拉松标准衡量 100 米短跑——标准本身即成为误判的来源。

## 格式分级

### 一级：长片/长篇小说（>5万字 / >90分钟）

- 适用完整的麦基标准
- 应全面运用五部分结构分析
- 人物应有弧光、欲望层次、真人物揭示
- 场景应有递进式价值转变序列
- 对白应充分实现三功能

### 二级：中篇/中视频/电视剧单集（1-5万字 / 20-60分钟）

- 结构标准适度放宽：允许简化版的三部分结构
- 人物弧可以聚焦于一个关键转变
- 次情节可以有但非必需
- 场景序列可以较短（2-3 个场景即可构成序列）

### 三级：极短篇/闪小说/短片（<1万字 / <20分钟）

- **主导标准不再是麦基的长片框架，而是爱伦·坡的"单一效果"原则** + 麦基的"价值转变"概念
- 分析重心从"完整的叙事结构"调整为"单一效果是否实现"

## 极短篇/短片各维度标准调整

### 故事结构

| 长片标准 | 极短篇调整 |
|----------|-----------|
| 必须包含完整的五部分结构 | 以"一个价值转变 + 一个关键时刻"为最小单位。允许省略部分结构环节 |
| 激励事件必须沉重打破平衡 | 激励事件可以是微妙的——一次见面、一句暗示、一个反常细节 |
| 危机必须是两难选择 | 危机可以被压缩入反转之中。"危机本身可能就是结局" |
| 高潮必须是所有线索的汇聚 | 高潮聚焦于*一个*核心效果即可 |
| 结尾需要余韵 | 结尾可以是一个意象、一个动作、一句话——点到即止是美德 |

**核心判断标准：** 这是否是爱伦·坡意义上"能够在一个坐着的时间里读完并产生单一效果"的作品？如果是，结构框架可以从宽评判。

### 人物设计

| 长片标准 | 极短篇调整 |
|----------|-----------|
| 必须区分人物塑造与真人物 | 允许"面具永远不被揭开"——画像式人物（type character）是一种刻意选择而非失败 |
| 必须有完整的人物弧光 | 人物可以不发生转变，甚至人物不需要"弧"。在闪小说中，揭露一个状态、一个情境、一个瞬间可以优于铺设一段弧线 |
| 自觉欲望 vs 不自觉欲望 | 在空间有限时，一层欲望就足够。不自觉欲望的缺失不是缺陷 |
| 主人公必须做选择 | 旁观者/透镜型叙事者是一种经典装置。不要将其自动标记为"被动主人公"的问题 |
| 对抗力须在三层次上设置 | 一个层次（如人与社会/人与情境）在短篇中已然足够 |

**透镜型叙事者判定：** 当叙事者主要功能是"观察窗口"、而焦点人物（Focal Character）承担故事的力量时，不应以"叙事者缺少欲望"为由批评，而应判断：焦点人物通过叙事者的眼睛呈现得是否有效？这个窗口是否开启了读者无法自行获得的理解角度？

### 对白与文本

| 长片标准 | 极短篇调整 |
|----------|-----------|
| 每句对白须服务于三功能之一 | 允许高度符号化/形式化的对白——"这颗是坏弹"式的单一功能对白在短篇中可以一击制胜 |
| 不同人物须有明显不同的声音 | 极短篇中可能只有1-2个说话人物，这一标准从宽评判 |
| 必须"展示多于讲述" | 极短篇中有限的"讲述"可以是有效的压缩技术。关键在于整体阅读体验是否由读者自己完成 |
| 叙述节奏须贴合情感节奏 | 极短篇中的重复（如"咔哒"排比）是合法的修辞技术——它不是冗余，而是强化单一效果的装置 |

### 类型与体裁

| 长片标准 | 极短篇调整 |
|----------|-----------|
| 须满足所有类型约定 | 极短篇可以聚焦于类型约定中的*一个核心约定*，其余的可以被省略 |
| 创新须在类型内寻找新视角 | 极短篇允许用"经典套路 + 精准执行"作为类型体验。不需要颠覆——完全履行约定就足够 |
| 类型混搭须谨慎 | 极短篇允许更自由的类型边界，因为读者不会在几千字内形成坚固的类型预期 |

### 场景设计

| 长片标准 | 极短篇调整 |
|----------|-----------|
| 每个场景必须有价值转变 | 在极短篇中，唯一的那个场景可能从头到尾积累张力——价值的转变可以发生在**最后一刻**（甚至场景之后）。这种"延迟释放"是一类有效的短篇技巧 |
| 节拍须递进升级 | 节拍可以重复/循环——循环本身可以是制造单一效果的装置（催眠感、紧张感的累积） |
| 转折点须既意外又合情理 | 极短篇的转折点就是全篇的高潮。它不需要在中间铺设小型的"场景内转折" |
| 场景须有因果连接 | 极短篇可能只有一个场景——这个标准不适用 |

## 极短篇评价的替代核心标准

当分析极短篇/闪小说时，以下标准应替代长片的"五部分结构完整性"：

1. **单一效果的清晰度** — 一个读完这篇作品后读者获得了什么不可替代的印象？
2. **经济的魅力** — 是否在极小的空间里创造了超出其篇幅的情感/智识密度？
3. **结尾的力量** — 最后的节拍是否让之前的一切获得了新的意义？
4. **装置的完整** — 有没有任何元素浪费了？有没有什么不在它该在的位置？
5. **重读价值** — 知道了结尾之后重读，作品是否提供了不同的体验？

## 分析时的标注规范

当判定作品为极短篇/短片时，在分析的每个维度中，先做以下声明：

```
> [格式声明] 本分析采用极短篇/短片标准，以下维度均已根据篇幅进行调整。
> 参考：references/format-adaptation.md
```

这确保了用户明白你使用的不是"放水"的标准，而是**适配格式的合理标准**。

## 极短篇常见误区（不分析这些问题）

以下是可以**优先跳过**的分析点，因为它们不适用于压缩格式：

- "人物没有完整的欲望层次" → 极短篇合理
- "只有一个场景/单一线索" → 极短篇合理
- "没有次情节" → 极短篇合理
- "对抗力层次不够丰富" → 极短篇合理
- "结尾过于开放" → 对于极短篇，这往往是有意为之

将分析火力集中在**那些在同格式作品中确实说明强度不够的地方**，而不是"没有达到另一类格式的标准"。
