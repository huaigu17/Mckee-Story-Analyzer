# 故事结构 (Story Structure) - 麦基理论参考

## 核心框架：五部分结构

### 1. 激励事件 (The Inciting Incident)

- 定义：彻底打破主人公生活平衡的事件，使其必须做出无可回避的抉择
- 判定标准：
  - 是否打乱了主人公的常规生活？
  - 是否激发了主人公内心一个自觉的欲望？
  - 是否创造一个"鸿沟"——期望与结果之间的裂痕？
- 位置：应尽早出现，但必须建立在充分铺垫之后。通常在前 1/4 处
- 常见错误：
  - **延迟激励事件**：观众/读者在等到真正的事件前失去耐心
  - **弱激励事件**：不足以迫使主人公采取行动，理由不充分
  - **伪装激励事件**：看似打破平衡，实际对主人公影响甚微

### 2. 进展纠葛 (Progressive Complications)

- 定义：随着主人公追寻欲望，遭遇越来越大的阻力，鸿沟不断扩大
- 核心原理——**鸿沟原理 (The Gap)**：
  - 主人公采取行动 → 预期一个结果
  - 现实返回一个不同的结果（鸿沟）
  - 主人公必须调整策略 → 再次行动
  - 循环往复，冲突升级
- 检验标准：冲突是否在逐层加深？主人公付出的代价是否越来越大？
- 常见错误：
  - **平行冲突**：冲突重复同一层次，没有质变
  - **缺乏因果链**：事件之间没有"因为 A 所以 B"的逻辑连接
  - **填充场景**：场景存在仅为过渡，没有任何价值转变

### 3. 危机 (Crisis)

- 定义：主人公面临最艰难的两难抉择，是故事的"必然场景"
- 特征：
  - 两个选择都不是好的选择（真正的两难）
  - 选择将决定高潮的方向
  - 是人物揭示的核心时刻——选择暴露真正的人格
- 判定标准：主人公是否面临一个"不得不做"的抉择？这个抉择是否有分量？
- 常见错误：
  - **假两难**：其中一个选项明显是正确答案
  - **无代价危机**：选择不需要主人公付出真正代价
  - **旁落危机**：危机不发生在主人公身上，或主人公没有选择权

### 4. 高潮 (Climax)

- 定义：主人公做出危机抉择后，展开的行动所带来的不可逆转的结果
- 特征：
  - 价值发生根本性、不可逆转的转变
  - 所有线索在此汇聚
  - 满足观众/读者的核心期待
- 检验标准：
  - 高潮是"不可避免的"（回溯能发现必然性）
  - 高潮是"出人意料的"（看的时候想不到）
  - 价值的转变是清晰的、有重量的
- 常见错误：
  - **虎头蛇尾**：高潮力度小于之前所有纠葛
  - **假高潮**：看似高潮，实则没有发生真正的价值转变
  - **deus ex machina**：靠外力解决，而非主人公自身行动

### 5. 结局 (Resolution)

- 定义：高潮之后的余波，展示新平衡的建立，释放剩余情感
- 功能：
  - 展示净效应（高潮带来的具体变化）
  - 闭环所有次情节
  - 释放剩余情感张力
- 原则：结局应简洁，不以拖沓消解高潮的力量
- 常见错误：
  - **过度解释**：不信任观众/读者，反复说明已表达的内容
  - **冗长拖拉**：高潮之后故事还在继续讲，削弱冲击力
  - **缺失结局**：戛然而止导致不完整感

## 主情节与次情节 (Main Plot & Subplot)

- 主情节：主人公追求核心欲望的主线
- 次情节：支撑主情节的辅助线
- 次情节的功能：
  - 强化主情节的主题
  - 提供对照（照镜子）
  - 增加复杂性（主人公生活的多重维度）
  - 制造反讽（次情节结果与主情节形成对比）
- 常见错误：次情节与主情节无关，未能服务于主题

## 三幕结构 (Three-Act Structure)

- 第一幕：铺垫 + 激励事件
- 第二幕：进展纠葛 + 危机
- 第三幕：高潮 + 结局
- 麦基态度：三幕是一种有用但不严格的工具，本质是"开端-发展-结局"的自然节奏

## 结构分析检查清单

- [ ] 激励事件是否足够有力？是否真正打破平衡？
- [ ] 鸿沟是否在每一轮行动中加深？
- [ ] 危机是否构成真正的两难选择？
- [ ] 高潮是否不可逆转且同时具有必然性与意外性？
- [ ] 结局是否简洁有力？
- [ ] 主情节与次情节是否有机相连？
- [ ] 是否有不必要的场景（没有价值转变的场景）？
